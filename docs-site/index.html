Swarm Communication Protocol

A decentralized, AI-native communication protocol for the next generation of applications. Build agents, swarms, and secure P2P messaging with cryptographic identity and end-to-end encryption.

Quickstart

1. Start a Relay Node

# Clone and build the relay
git clone https://github.com/swarmchat/scp-relay
cd scp-relay && go build -o scp-relay .

# Start the relay
./scp-relay --listen /ip4/0.0.0.0/udp/9090/quic-v1 --http :8080

2. Create an Identity

# Using CLI
cd scp-cli && go build -o scp-cli .
./scp-cli identity create --label personal --nickname "Alice"

# Using Python SDK
from scp_sdk import SCPClient
client = SCPClient("http://localhost:8080")

3. Build an Agent (Python)

from scp_sdk import SCPAgent, on_message, MessageType

class EchoBot(SCPAgent):
    @on_message(MessageType.TEXT)
    def echo(self, envelope, payload):
        self.reply(envelope, f"Echo: {payload['body']}")

bot = EchoBot(client=client, did="did:key:z6Mk...", name="EchoBot")
bot.run_webhook(port=5000)

4. Send a Message

# CLI
./scp-cli message send --to did:key:z6Mk... --text "Hello Swarm!"

# Python
client.send_text(from_did=my_did, to_did=peer_did, text="Hello!")

Relay API Reference

All endpoints return JSON. Base URL: http://your-relay:8080

MethodPathDescription
GET /health Health check with node status, PeerID, version
GET /node/info Node information: PeerID, addresses, protocols
POST /prekey Upload a prekey bundle (identity key, signed prekey, one-time prekeys)
GET /prekey/{did} Get a prekey bundle for a specific DID
POST /message Store an offline message (encrypted envelope with TTL)
GET /sync/{did} Sync offline messages for a DID (with since/limit params)

Protocol Specification

§3 Network Architecture

P2P overlay with QUIC transport, Kademlia DHT, GossipSub pub/sub, and Circuit Relay v2 for NAT traversal.

§4 Identity & Encryption

BIP39 mnemonics, did:key identifiers, X3DH key agreement, Double Ratchet for 1:1, Sender Key for groups.

§5 Message Protocol

25 message types across 6 categories: basic messages, group management, agent semantics, swarm coordination, and system control.

§6 Swarm Intelligence

Agent capability cards, task assignment and claiming, sub-swarm formation, and consensus mechanisms.
