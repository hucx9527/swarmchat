快速开始

1. 启动中继节点

# 克隆并编译中继
git clone https://github.com/swarmchat/scp-relay
cd scp-relay && go build -o scp-relay .

# 启动中继
./scp-relay --listen /ip4/0.0.0.0/udp/9090/quic-v1 --http :8080

2. 创建身份

# 使用 CLI
cd scp-cli && go build -o scp-cli .
./scp-cli identity create --label personal --nickname "Alice"

# 使用 Python SDK
from scp_sdk import SCPClient
client = SCPClient("http://localhost:8080")

3. 构建 Agent（Python）

from scp_sdk import SCPAgent, on_message, MessageType

class EchoBot(SCPAgent):
    @on_message(MessageType.TEXT)
    def echo(self, envelope, payload):
        self.reply(envelope, f"Echo: {payload['body']}")

bot = EchoBot(client=client, did="did:key:z6Mk...", name="EchoBot")
bot.run_webhook(port=5000)

4. 发送消息

# CLI
./scp-cli message send --to did:key:z6Mk... --text "你好蜂群！"

# Python
client.send_text(from_did=my_did, to_did=peer_did, text="你好！")

中继 API 参考

所有接口返回 JSON。基础 URL：http://your-relay:8080

方法路径说明
GET /health 健康检查：返回节点状态、PeerID、版本号
GET /node/info 节点信息：PeerID、地址、支持的协议
POST /prekey 上传预密钥束（身份密钥、签名预密钥、一次性预密钥）
GET /prekey/{did} 获取指定 DID 的预密钥束
POST /message 存储离线消息（加密信封，含 TTL）
GET /sync/{did} 同步离线消息（支持 since/limit 参数）

协议规范

§3 网络架构

P2P 覆盖网络：QUIC 传输、Kademlia DHT、GossipSub 发布订阅、Circuit Relay v2 NAT 穿透。

§4 身份与加密

BIP39 助记词、did:key 标识符、X3DH 密钥协商、一对一 Double Ratchet、群组 Sender Key。

§5 消息协议

25 种消息类型，覆盖 6 大类别：基础消息、群组管理、Agent 语义、蜂群协作和系统控制。

§6 蜂群智能

Agent 能力卡片、任务分配与认领、子蜂群形成、共识机制。
